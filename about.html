Current API info

There are currently 5 endpoints:

  1. To see all current users: /users
  2. To see a particular user: /users/[user id]
  3. To create a user: curl --data "name=Elaine&email=elaine@example.com" http://localhost:3000/users
  4. To update a user: curl --data "name=Elaine&email=elaine@example.com" http://localhost:3000/users
  5. To delete a user: curl -X "DELETE" http://localhost:3000/users/[user id]
